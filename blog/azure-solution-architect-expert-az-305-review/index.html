Azure Solution Architect Expert (AZ-305) Review

March 26, 20242 minutes

My experience with studying and taking Azure Solution Architect Expert (AZ-305) certification.

Introduction

Around a month ago I obtained the Azure Security Engineer Associate (AZ-500) certification and quickly after passing the certification I decided to get the Azure Solution Architect Expert (AZ-305) certification since the knowledge obtained from AZ-104 and AZ-500 correlates with AZ-305.

Taking an break would be an bad decission as it’s easy to forget the things I learned throughout AZ-104 and AZ-500.

Preperation

As always I first read through the Microsoft Learn: Designing Microsoft Azure Infastructure Solutions course. While going through the course I took notes and created graphs to further increase my understanding about Azure resources and features. Here’is a overview of the graphs which I made during my note taking:

Azure Service Bus

Azure Event Hub

I also spent a lot of time going through my notes and researching the Azure resources and features which I wasn’t familiar with such as:

Keywords
  • Azure Service Fabric
  • Azure Cache for Redis
  • Azure Application Insights
  • Azure Data Lake
  • Azure Data Factory
  • Azure IT Service Management Connector
  • Azure NetApp
  • Azure Diagnostic Extension
  • Azure HDInsights
  • Azure SQL Analytics

Once I was finished with researching the Azure resources and features, I spent decent amount of time re-calling the things I learned throughout AZ-104, AZ-500, and AZ-305 course materials.

Exam

Today at 20:00, I was going through my notes and re-calling the things which I forgot. Out of boredom I decided to check if there were any AZ-305 seats available for today and I saw one was available at 20:30 so I quickly purchased it and joined the AZ-305 exam.

My adrenaline was extremely high during the AZ-305 exam as I was taking the exam without being prepared for it which is abnormal for me. Here’s a overview of my experience with AZ-305 exam:

  • Multiple of choices were difficult but I quickly went through all of these questions as my adrenaline was high.
  • Non-skipeable multiple of coices wasn’t going my way as I was speed reading through some of the questions and probably got one or two questions wrong.
  • Case study was difficult but I was confident which the answers which I selected as I have been researching Azure for a long-time now.

Out of nowhere before I knew it I finished 59 questions within 30 minutes. Once I sent the AZ-305 exam in my adrenaline was even higher but once I saw the 780 score I was reliefed and extremely happy as I managed to pass the Azure Solution Architect Expert (AZ-305) certification.
